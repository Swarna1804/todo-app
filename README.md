# 📝 Todo List App

A todo list app is basically a task management tool. Its core purpose is to let users create, track, update, and complete tasks in a structured way.

---

## 🛠️ Tech Stack
1. HTML  
2. CSS  
3. TypeScript  

---

## 🔑 Core Functionalities

➕ **Add tasks** – create new to-do items  
✏️ **Edit tasks** – modify existing tasks  
✅ **Mark as completed** – track progress  
🗑️ **Delete tasks** – remove unwanted tasks  
📋 **View tasks list** – see all pending and completed work  

---

## 🎯 Why we use a Todo List App

A todo list is used because it helps people manage time and work better.

---

## 🚀 Main Reasons

🧠 **Reduces mental load**  
You don’t have to remember everything in your head.

⏰ **Improves productivity**  
Helps you focus on important tasks instead of forgetting them.

📌 **Better organization**  
Tasks are structured in one place instead of scattered notes.

🎯 **Clear priorities**  
You can decide what to do first, next, and later.
